== RAW — ตัวอย่าง UI Components สำหรับอ้างอิงการสร้าง Screen ==
ระบบ: 6_5
วันที่บันทึก: 2026-04-16

---

[ Views/Forms ]
ไฟล์ที่มีใน Views/Forms:
- BasicInputs.cshtml       → input text, email, password, number, textarea พื้นฐาน
- CustomOptions.cshtml     → checkbox, radio แบบ custom style
- Editors.cshtml           → rich text editor (WYSIWYG)
- Extras.cshtml            → input พิเศษ เช่น input with icon, floating label
- FileUpload.cshtml        → อัปโหลดไฟล์ single/multiple, drag & drop
- InputGroups.cshtml       → input พ่วง prefix/suffix (เช่น ฿, %, @)
- Pickers.cshtml           → date picker, time picker, color picker
- Selects.cshtml           → select box, multi-select, select2 (searchable dropdown)
- Sliders.cshtml           → range slider
- Switches.cshtml          → toggle switch

ใช้อ้างอิงตอนสร้างหน้าจอที่มีฟอร์ม เช่น หน้ายื่นคำขอ

---

[ Views/Icons ]
ไฟล์ที่มีใน Views/Icons:
- RiIcons.cshtml           → ไอคอนจาก Remix Icon ทั้งหมด ดูชื่อ class ที่นี่

ใช้อ้างอิงตอนเลือกไอคอนให้ปุ่ม, เมนู, สถานะ

---

[ Views/Modals ]
ไฟล์ที่มีใน Views/Modals:
- Examples.cshtml          → modal ขนาดต่างๆ (sm, md, lg, xl, fullscreen)
                             modal แบบมี form ข้างใน
                             modal แบบ confirm (ยืนยัน/ยกเลิก)
                             modal แบบ scrollable

ใช้อ้างอิงตอนสร้าง popup ยืนยันการอนุมัติ/ปฏิเสธ

---

[ Views/Tables ]
ไฟล์ที่มีใน Views/Tables:
- Basic.cshtml              → ตาราง HTML ธรรมดา striped, bordered, hover
- DatatablesBasic.cshtml    → DataTables พื้นฐาน: sort, search, pagination
- DatatablesAdvanced.cshtml → DataTables advanced: fixed column, responsive, scroll
- DatatablesExtensions.cshtml → export Excel/PDF/CSV, print, column visibility

ใช้อ้างอิงตอนสร้างหน้ารายการ เช่น คิวงาน, รายการผู้ใช้, audit log

---

[ Views/Ui ]
ไฟล์ที่มีใน Views/Ui:
- Accordion.cshtml          → accordion / collapse panel
- Alerts.cshtml             → alert box: success, warning, danger, info
- Badges.cshtml             → badge สำหรับแสดงสถานะ จำนวน
- Buttons.cshtml            → button ทุกขนาดและทุก variant
- Carousel.cshtml           → image carousel / slider
- Collapse.cshtml           → ซ่อน/แสดงเนื้อหา
- Dropdowns.cshtml          → dropdown menu ทุกแบบ
- Footer.cshtml             → footer layout
- ListGroups.cshtml         → list group สำหรับแสดงรายการ
- Modals.cshtml             → modal พื้นฐานจาก Bootstrap
- Navbar.cshtml             → top navigation bar
- Offcanvas.cshtml          → sidebar แบบ slide-in
- PaginationBreadcrumbs.cshtml → pagination และ breadcrumb
- Progress.cshtml           → progress bar
- Spinners.cshtml           → loading spinner
- TabsPills.cshtml          → tabs และ pills navigation
- Toasts.cshtml             → toast notification (แจ้งเตือนมุมจอ)
- TooltipsPopovers.cshtml   → tooltip และ popover
- Typography.cshtml         → หัวข้อ, paragraph, text style

ใช้อ้างอิงตอนออกแบบ layout และ component ต่างๆ ในแต่ละหน้าจอ
